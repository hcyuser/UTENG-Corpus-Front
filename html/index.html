Rate My professor Corpus System

Introduction

This is a corpus database collecting the comments and other attributes from Rate My Professors. It records the comments from Jan. 1999 to Jan. 2018, which are curled from 25 Jan. 2018 to 25 Feb. 2018 by Google Cloud Platform (GCP). The establishers of this database are Mei-ching Ho and HUANG, CHIH-YANG. This project is sponsored by Ministry of Science and Technology (MOST), TAIWAN(R.O.C.).

Query Option

  • Overall Query: for researchers to peep into the data

  • The Number of Professors Query: query the number of professors in an university

  • Advanced Query: to query specified overall quality which the comments correspond to the quality

  • Advanced Query with Location: to query specified overall quality which the comments correspond to the quality with the condition of location

  • Advanced Query with Customized Quality: to query specified overall quality which the comments correspond to the customized quality
